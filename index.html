Currency Follower

This handy Java application uses Jsoup and log4j to follow currency changes on a specyfic website and the present it in a comparison type of form. Following changes on most recognized currencies and checking for rises / falls comapring to first day of current month. Computional complexity is way to high, but concerning non-changable, small size of data it is sufficient for now.
